2. apa makna lagu dhyo haw kecewa​


Lagu "Kecewa" dari Dhyo Haw menggambarkan perasaan kecewa seseorang terhadap orang yang dicintainya. Dalam lagu ini, Dhyo Haw mengekspresikan perasaan kesedihan dan kekecewaan seseorang yang merasa telah dikhianati oleh orang yang dianggap penting dalam hidupnya. Namun, meskipun merasa hancur dan putus asa, lagu ini memberikan pesan untuk tetap berusaha bangkit dari kekecewaan dan melangkah maju. Dalam keseluruhan lagu, Dhyo Haw menyampaikan pesan bahwa kekecewaan adalah bagian dari hidup dan kita harus belajar untuk menerima kekecewaan, mengambil pelajaran dari pengalaman tersebut, dan tumbuh menjadi lebih kuat.

Dhyo Haw merilis album debutnya yaitu Always Positive pada tahun 2013. 5. angka kunci pada tombol numerik pada keyboard adalah​


Jawaban:

Angka kunci pada tombol numerik pada keyboard adalah 0 sampai dengan 9, +, - , * , dan /.

Penjelasan:

Angka numerik adalah angka yang terdapat pada keyboard dengan tujuan sebagai alat untuk menginput data berupa angka. dan juga beberapa fungsi matematika seperti + (tambah), - (kurang), * (Kali) dan / (Bagi). 28. Jelas 5 Kata kunci Dari keyboard


Jawaban:

Ctrl + C berfungsi untuk melakukan copy pada obyek yang dipilih (memilih obyek, fungsinya hampir sama seperti Ctrl + X namun shortcut ini berfungsi untuk memilih).

Ctrl + S berfungsi untuk melakukan save pada program yang sedang diproses, biasanya berlaku pada microsoft word, microsoft powerpoint, notepad, paint, ataupun microsoft excel.

Ctrl + A berfungsi untuk memilih atau select seluruh obyek pada program yang sedang diproses.

Ctrl + X berfungsi untuk melakukan cut pada obyek yang dipilih (memotong obyek, fungsinya hampir sama seperti Ctrl + C namun shortcut ini berfungsi untuk memotong).

Ctrl + V berfungsi untuk melakukan paste obyek yang telah di cut ataupun copy

Jawaban:

1. Fungsi Tombol Esc

Tombol Escape untuk membatalkan perintah yang sudah dilakukkan. Jika kita menekan tombol ini maka perintah atau dialog yang kita masukkan akan dibatalkan.

2. Fungsi Tombol Tab

Tombol Tab berfungsi jika ditekan akan memindahkan cursor ke arah kanan dengan satu tabulansi.

3. Fungsi Tombol Caps Lock 

Jika menekan tombol ini maka lampu indikator caps lock pada keyboard menyala, hal ini menandakan bahwa anda menggunakan mode semua huruf besar atau semua huruf kapital, jika anda menekan lagi dan lampu indikator mati maka dalam mode seperti biasa atau semua huruf kecil.

4. Fungsi Tombol Shift

Fungsinya yaitu untuk membuat huruf besar secara manual (dengan menekan shift+a = A), dan untuk memasukkan input yang diatas angka 1,2,3 dan seterusnya (contoh : shift+1 = ! , shift+/ = ?).

5. Fungsi Tombol Ctrl

Khusus untuk tombol ini fungsinya itu untuk menyisipkan perintah-perintah pintas. Contohnya untuk mencopy  file tinggal tekan ctrl+c.

6. Fungsi Tombol Alt

Fungsinya sama seperti tombol Ctrl yaitu untuk menyisipkan perintah secara pintas. Contohnya alt+F4, itu untuk menutup program yang sedang dijalankan.

7. Fungsi Tombol Enter

Sebagai perintah "ok" yang diberikan oleh program pada komputer dan untuk membuat paragraf baru.

8. Fungsi Tombol Back Space 

Fungsinya untuk mendelete satu huruf yang berada disebelah kiri kursor. Jika kita kombinasikan dengan tombol ctrl, ctrl+backspace maka anda menghapus satu kata disebelah kiri kursor.

9. Tombol Delete

Tombol ini sama seperti tombol backspace, bedanya itu jika tombol delete menghapus satu huruf di sebelah kanan kursor.

10. Tombol Macro 

Macro ini jika ditekan akan muncul seperti ini "\".

11. Tombol Insert 

Untuk menyisipkan karakter di posisi kursor.

12. Tombol Home

Fungsinya untuk langsung menuju bagian atas halaman atau lembar kerja.

13. Tombol End

Fungsinya untuk langsung menuju bagian bawah halaman atau lembar kerja.

14. Tombol Page Up

Fungsinya untuk menggerakkan baris satu demi satu layar ke atas.

15. Tombol Page Down

Fungsinya untuk menggerakkan baris satu demi satu layar ke bawah.
