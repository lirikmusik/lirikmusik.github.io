Jawabannya adalah WALKED ✔️

PEMBAHASANPast Tense

Digunakan untuk menyatakan Kegiatan di masa lampau. Ciri waktu lampau:

=> yesterday

=> last month ✔️

=> two days ago

=> This morning

Rumus Past Tense

(+) S + V2

(-) S + did not + V1

(?) Did + S + V1?

KESIMPULAN

We WALKED to the jungle last month

=> keterangan lampau “last month”=> Berarti menggunakan Verb-2 “walked”
