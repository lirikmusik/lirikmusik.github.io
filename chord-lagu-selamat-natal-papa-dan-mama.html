6. Bahasa arab mama dan papa


Bahasa arab mama dan papa adalah أُمِّي dan أبي

Pembahasan

Di dalam Bahasa Arab, kita dapat memanggil mama dengan sebutan أُمِّي ( ummi ) dan memanggil papa dengan sebutan أبي ( abi )

أُمِّي ( ummi )

         Artinya adalah ibuku. Seorang ibu mengurus rumah tangga dan mengurus suami dan anak-anaknya selama mereka berada di rumah.

أبي ( abi )

         Artinya adalah ayahku. Seorang ayah mencari nafkah yang halal nan baik untuk anak-anak dan istrinya agar dapat memenuhi kebutuhan mereka sehari-hari

Terkadang ada seseorang yang memanggil mereka itu Ummu dan Abu. Mereka memiliki perbedaan dengan ummi dan abi.

ام ( ummu )

    Mereka memakai ini untuk memanggil seorang ibu dari anak tersebut. Misalkan ibu tersebut memiliki anak bernama Nurul. Maka, orang memanggil nya adalah Ummu Nurul.

اب ( abu )

   Mereka memakai ini untuk memanggil seorang ayah dari anak tersebut. Misalkan ayah tersebut memiliki anak bernama Harun. Maka, orang memanggil nya adalah Abu Harun.

32. Quiz Pertama! Spesial Hari Natal✨Soal → Apa itu Natal?_____Selamat Hari Natal....​


Natal adalah hari raya umat Kristiani untuk memperingati hari  kelahiran Yesus Kristus. Umat Nasrani merayakan hari Natal secara khidmat dan kebesaran baik di dalam gereja ataupun di rumah-rumah.

Menurut bahasa latin, Natal berarti lahir. Sedangkan menurut istilah, Natal berarti upacara yang dilakukan oleh orang kristen untuk memperingati hari kelahiran Isa al-Masih, yang mereka sebut dengan Tuhan Yesus.

Tradisi yang biasa dilakukan selama perayaan Natal berlangsung adalah gereja-gereja dihias dengan mewah dan semegah mungkin. Orang-orang Kristiani saling bertukar kado dan saling memberi hadiah-hadiah, menghiasi rumah mereka dengan daun holly, mistletoe, dan pohon Natal, dan lain sebagainya.

Penyelesaian

Diketahui :

Keliling = 72 m

Panjang = ( 5x - 1 ) m

Lebar = ( 2x + 2 ) m

Ditanya :

P sesungguhnya

L sesungguhnya

Jawab :

Cari nilai x

K = 2( p + l )

2( ( 5x - 1 ) + ( 2x + 2 ) ) = 72

2( ( 5x + 2x ) + ( - 1 + 2 ) ) = 72

2( 7x + 1 ) = 72

14x + 2 = 72

14x = 72 - 2

14x = 70

x = 70 ÷ 14

x = 5

Panjang Sesungguhnya

P = ( 5x - 1 )

P = ( 5(5) - 1 )

P = 25 - 1

P = 24 m

Lebar Sesungguhnya

L = ( 2x + 2 )

L = ( 2(5) + 2 )

L = ( 10 + 2 )

L = 12 m

Kesimpulan :
