13. Tuliskan syntax Php untuk menghapus data siswa berdasarkan ID siswa​


Jawaban:

berikut code nya :

<?php

// Koneksi ke database

$conn = mysqli_connect("localhost", "username", "password", "nama_database");

// Menangkap ID siswa yang akan dihapus dari parameter GET

$id_siswa = $_GET["id"];

// Membuat query untuk menghapus data siswa berdasarkan ID

$query = "DELETE FROM siswa WHERE id_siswa = $id_siswa";

// Menjalankan query untuk menghapus data

if (mysqli_query($conn, $query)) {

echo "Data siswa berhasil dihapus";

} else {

echo "Gagal menghapus data siswa: " . mysqli_error($conn);

}

// Menutup koneksi ke database

mysqli_close($conn);

?>

Penjelasannya :

1. Pertama-tama kita melakukan koneksi ke database dengan menggunakan fungsi `mysqli_connect()` . Di sini, kamu harus mengganti "localhost", "username", "password", dan "nama_database" dengan nilai yang sesuai untuk koneksi ke database kita.

2. menangkap ID siswa yang akan dihapus dari parameter GET dengan menggunakan variabel `$_GET["id"]`.

3. Setelah itu, kita membuat query SQL untuk menghapus data siswa berdasarkan ID menggunakan perintah `DELETE FROM`. Di sini, kita menggunakan variabel `$id_siswa` untuk menggantikan nilai ID siswa yang kita tangkap sebelumnya.

4. Kita mengeksekusi query menggunakan fungsi `mysqli_query()`. Jika query berhasil dijalankan, kita akan menampilkan pesan "Data siswa berhasil dihapus". Jika tidak, kita akan menampilkan pesan error menggunakan fungsi `mysqli_error()`.

5. Terakhir, kita menutup koneksi ke database menggunakan fungsi 'mysqli_close()'.

Sekian semoga membantu

Jawaban:

<?php

// koneksi ke database

$servername = "localhost";

$username = "username";

$password = "password";

$dbname = "myDB";

$conn = mysqli_connect($servername, $username, $password, $dbname);

// periksa koneksi

if (!$conn) {

 die("Koneksi gagal: " . mysqli_connect_error());

}

// ambil ID siswa yang akan dihapus dari parameter URL

$id = $_GET['id'];

// query untuk menghapus data siswa berdasarkan ID siswa

$sql = "DELETE FROM siswa WHERE id=$id";

// eksekusi query

if (mysqli_query($conn, $sql)) {

 echo "Data siswa berhasil dihapus";

} else {

 echo "Error: " . $sql . "<br>" . mysqli_error($conn);

}

// tutup koneksi

mysqli_close($conn);

?>

Penjelasan:

Koneksi ke database diinisialisasi dengan variabel $servername, $username, $password, dan $dbname.Koneksi ke database dilakukan dengan fungsi mysqli_connect().ID siswa yang akan dihapus diambil dari parameter URL dengan variabel $id.Query untuk menghapus data siswa dengan ID $id dibuat dengan variabel $sql.Query dieksekusi dengan fungsi mysqli_query().Jika query berhasil dieksekusi, pesan berhasil dicetak ke layar. Jika tidak, pesan error dicetak ke layar.Koneksi ditutup dengan fungsi mysqli_close().

16. tabel berikut menampilkan data dua trafo ideal berdasarkan data pada tabel diatas nilai K dan L berturut turut adalah


Jawaban:

Nilai K sebesar 2 A dan nilai L adalah 30 Volt (Opsi B)

Penjelasan:

Diketahui :

Travo 1 :

NP = Lilitan Primer = 20

NS = Lilitan sekunder = 30

VP = Tegangan Primer = 200 Volt

VS = Tegangan Sekunder = 300 Volt

IP = Arus Primer = 3 A

Travo 2  :

NP = Lilitan Primer = 40

NS =  Lilitan sekunder = 30

VP = Tegangan Primer = 40 Volt

IS = Arus sekunder = 0,75 A

IP = Arus Primer = 1 A

Ditanyakan :

K = IS pada Travo 1 = ...

L  = VS = Pada Travo 2 = ...

Penyelesaian :

[tex]\frac{VP}{VS}=\frac{NP}{NS}=\frac{IS}{IP}[/tex]

Pada Travo 1 :

[tex]\frac{200}{300}=\frac{IS}{3}[/tex]

300 x IS = 200 x 3

IS = 600/300

IS = 2 A

Nilai K adalah 2 A

Pada Travo 2 :

[tex]\frac{40}{VS}=\frac{40}{30}[/tex]

40 x VS = 30 x 40

VS = 1200/40

VS = 30 Volt

nilai L adalah 30 Volt

OR merupakan syntax search engine untuk mencari salah satu kata tertentu, atau keduanya atau lebih. Sama dengan OR, simbol vertikal (|) ini bisa digunakan jika caps lock rusak.
