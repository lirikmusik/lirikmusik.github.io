9. wo didi he wo mei mei xihuan heise de yifu artinya apa ?​


Arti dari bahasa mandarin diatas adalah :

我弟弟和我妹妹喜欢黑色的衣服

= wǒ dì dì hé wǒ mèi mèi xǐ huān hēi sè de yī fú.

= Adik [ laki-laki ] saya dan Adik [ Perempuan ] saya suka baju berwarna hitam.

Penjelasan :

Jika kita mengartikan secara satu persatu maka akan mendapatkan arti yaitu :

我弟弟 = wǒ dì dì = Adik laki laki saya

和 = hé = Dan

我妹妹 = wǒ mèi mèi = Adik perempuan saya

喜欢 = xǐ huān = suka atau menyukai

黑色 = hēi sè = warna hitam

的 = de = untuk menyatakan kepunyaan

衣服 = yī fú = Baju

Semoga bermanfaat

14. perbedaan bahasa mandarin bu you dan mei you​


Jawaban:

不 (bù) digunakan untuk menyatakan negasi di masa sekarang dan masa depan

不 (bù) umumnya digunakan untuk menyangkal suatu kegiatan yang tidak ingin Anda lakukan atau tidak bermaksud melakukannya (di masa depan). Jadi menyatakan hal-hal seperti "Saya tidak ingin pergi" atau "Saya tidak akan pergi" akan menggunakan 不 (bù).

没 (méi) untuk menyatakan negasi 有 (yǒu)

没 (méi) dapat digunakan untuk menyatakan negasi kata kerja 有 (yǒu) ("memiliki"; ada") sehingga artinya menjadi "tidak memiliki", "tidak ada". Anda hanya dapat menggunakan 没 (méi) untuk mengutarakan maksud ini. Secara tata bahasa tidak tepat menggunakan 不 (bù) untuk menyatakan negasi 有 (yǒu).
